# Virtual_Master_PlayStore_4.2.2
This repo/apk will install the Google Play Services, to android 4.2.2, since it had no root and GApps
This how i'm the first person to make this installer.
# INSTRUCTIONS
1) Download the .apk file in the releases page
2) Import the app by pressing Add Files (to import the apk, no need to install it, cuz importing it installs it)
3) Open the app, you will see this:
<img width="720" height="1640" alt="Image" src="https://github.com/user-attachments/assets/96797fd9-0989-4a91-820f-98bef0e669c5" />

4) Press button 1, then 2, and finally 3.
5) Log in (by opening the play store)
6) Finished!
# ADVISE
If you install the apps in random order, it wont work
# TESTED
Tested on Redmi 14C, as always, Virtual Master, running PixelOS 16.
